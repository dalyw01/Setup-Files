+-----------------------------------------------------+
Creating repo from local files to an online GitHub repo
+-----------------------------------------------------+
FIRST create the repo on github e.g - https://github.com/dalyw01/my_new_repo

mkdir project
cd project
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/dalyw01/my_new_repo.git
git push -u origin master

+------------------------------------------------------------------------------------------+
Downloading online repo code, setting up new branch from + commiting to the NEW branch
+------------------------------------------------------------------------------------------+

mkdir x
cd x

git init
git clone https://github.com/dalyw01/Google-Voice-Assistant-Testing

git branch will_branch
git checkout will_branch

touch 123.txt
git add . 

git commit -m "comment adding new file to branch"
git push origin will_branch

 * [new branch]      will_branch -> will_branch

Check github and you'll see a new branch has been created!

+-------------------------------------------+
Setting a new remote after breaking previous
+-------------------------------------------+

git remote set-url origin https://github.com/dalyw01/Scanner.git
git pull origin master
